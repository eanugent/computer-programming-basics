CPB-06 Conditional Statements

Press enter/return to start

Conditional statements are used to execute a block of code only when certain conditions are true. In this exercise we will be implementing the moveAvatar function which moves the avatar to a different location on the screen based on the parameters given.

Instructions

  1. Add conditional statements (with empty code blocks for now) to be executed in the following cases:
    • When the value of axis is 'x'
    • When the value of axis is 'y'
    • When the value of axis is neither 'x' nor 'y'
    (see the first note below)
  2. In the block that executes when axis is 'x', create a newX variable whose value is the return value of the getAvatarX() function plus the value of the spaces parameter.
  3. After your declaration of the newX variable, call the setAvatarX function with newX as its parameter.
  4. Use the previous two instructions to complete the block that executes when the axis is 'y' (changing X to Y as appropriate).
  5. In the block that executes when axis is neither 'x' nor 'y', report an error using the following line of code:

    throw 'axis is an invalid value'

  6. After the previously-defined conditional blocks, declare a variable named hitBlockIndex and set its value to the return value of the indexOfBlockHitByAvatar() function.
  7. Add a conditional statement (with an empty code block for now) that will execute its code block if the value of hitBlockIndex is greater than -1.
  8. In the code block, call the clearBlock function passing hitBlockIndex as its parameter.
  9. Also in this code block, call the playSound function passing 'beep.wav' as its parameter.

Notes

  • In Javascript, to test if a value is equal to another value, we use 3 equal signs as in the following:
    if(myVariable === 'this value'){ }
