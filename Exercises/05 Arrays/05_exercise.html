CPB-05 Arrays

Press enter/return to start

Arrays allow us to store a collection of items in a single structure. In this exercise we will be storing the colors and the target blocks in an array. This is a reorganization exercise so the program should run the same before and after our changes.

Instructions

  1. Inside of the run function, declare an array named colors.
  2. After the array declaration, use the push function to add the 3 colors that were being stored in the color1, color2, and color3 variables to the array.
  3. Remove the color1, color2, and color3 variables declarations
  4. Change the color1, color2, and color3 references in the changeAvatarColor function calls to reference the elements of the array instead.
  5. Declare a global array named blocks.
  6. Inside of the drawTargetBlocks function, instead of declaring 3 variables, push the block objects into the blocks array.
  7. Replace the 3 drawBlock (singular) function calls with a single call to the drawBlocks (plural) function passing the blocks array as the parameter.

Notes

  • An array can be declared using the following syntax:

    const myArray = []

  • Items can be added to an array using the push function:

    stringArray.push('String to add') objectArray.push({ propX: 100, propY: 200})

  • Elements in an array can be accessed by putting the index of the desired element in square brackets after the name of the array. The index of the first element is 0. So, to get the 3rd element from myArray we use the following:

    myArray[2]

  • For more information on Arrays in Javascript, see the W3 Schools reference.
