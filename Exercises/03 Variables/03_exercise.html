CPB-03 Variables

Press enter/return to start

Variables are named containers to store information in a computer program. In this exercise we will better organize our program by adding variables.

Instructions

  1. Declare a defaultSleepTime variable.
  2. Assign the value of 1000 to the defaultSleepTime variable.
  3. Declare a color2 variable and a color3 variable.
  4. Assign the values of the color2 and color3 variables to be colors of your choice.
  5. Assign the value of the global avatarColor variable to be a color of your choice.
  6. Use the defaultSleepTime variable to replace the 1000 in the sleep function calls.
  7. Use the color2 and color3 variables to replace the parameters in the 2nd and 3rd changeAvatarColor function calls.

Notes

  • All of your code changes should be inside of the run function.
  • Your code for instructions 1-5 should be before line 6.
  • In the programming language we're using (JavaScript), variables can be declared using the let keyword as on line 1. If we know the value we want to initially assign to a variable when it is declared, we can do the assignment at the same time as the declaration as on line 4. Finally, if we want to assign a value to a previously-declared variable (like instruction #5), the let keyword should not be used.
