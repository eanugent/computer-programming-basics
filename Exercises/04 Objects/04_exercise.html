CPB-04 Objects

Press enter/return to start

Objects store related values (and/or functions) for better sharing and processing of information in a computer program. In this exercise we will better organize our program by grouping avatar-related values into an object.

6 more functions are being defined in the starter code for our program. They are used to get and set the avatar's x, y, and color values. Currently, these functions use the avatarColor, avatarX, and avatarY variables. In this exercise, you will be changing these functions to use an object instead.

Instructions

  1. Replace the avatarColor, avatarX, and avatarY variable declarations (lines 1-3) with a declaration of an avatar variable.
  2. Replace the avatarX, avatarY, and avatarColor variable assignments with assignments of the x, avatarColor, and avatarColor properties of the avatarColor object.
