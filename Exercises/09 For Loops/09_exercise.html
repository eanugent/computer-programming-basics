CPB-09 For Loops

For Loops can simplify code that uses a While Loop in some scenarios. In this exercise we will refactor (see note below) the drawBlocks function using a For Loop.

Instructions

  1. Replace the code in the drawBlocks function with a for loop.

Notes

  • Refactoring is the process of restructuring code without changing its overall function.
  • A for loop consists of 3 expressions:

    for(BEFORE; WHILE; AFTER){ }

    • BEFORE: Executed before the block in the loop. Usually a declaration and initialization of the "iteration variable". For example: let index = 0
    • WHILE: The conditional statement that determines how long the block will be repeated. Usually a less-than expression with the iteration variable on the left. For example: index < myArray.length
    • AFTER: Executed after each iteration of the loop. Usually incrementing the iteration variable. For example: index++
  • For more information on While Loops in Javascript, see the W3 Schools reference.
