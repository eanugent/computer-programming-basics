CPB-03 Variables

Press enter/return to start

Variables are named containers to store information in a computer program. In this exercise we will better organize our program by adding variables.

Instructions

  1. Using the example on line 4, declare color2 and color3 variables and assign them to colors of your choice.
  2. Declare a defaultSleepTime variable assigned to the value of 1000 (quotes are not needed).
  3. In the run function, assign a color of your choice to the globally declared avatarColor variable (see the first note below).
  4. Use the color2 and color3 variables to replace the parameters in the 2nd and 3rd changeAvatarColor function calls.
  5. Use the defaultSleepTime variable to replace the parameter in the sleep function calls.

Notes

  • In the programming language we're using (Javascript), variables can be declared using the let keyword as on line 1. If we know the value we want to initially assign to a variable when it is declared, we can do the assignment at the same time as the declaration as on line 4. Finally, if we want to assign a value to a previously-declared variable (like instruction #3), the let keyword should not be used.
  • For more information on Variables in Javascript, see the W3 schools reference.
