CPB-08 Loops

Loops allow us to repeatedly execute a block of code as long as a certain condition is true. In this exercise, we will use a while loop to implement the drawBlocks function.

Instructions

  1. In the drawBlocks function (line 53), create a variable named index and initialize it to a value of 0.
  2. After the variable declaration, add a while loop that will execute as long as the value of index is less than the length property of the blocks array. The code block will be populated in the next step.
  3. Inside of the while loop, call the drawBlock function passing the blocks array's element at the value of index. See the first note below.
  4. After the function call, set the value of index to be its current value plus 1.

Notes

  • Elements in an array can be accessed by putting the index of the desired element in square brackets after the name of the array. The index of the first element is 0. So, to get the 3rd element from myArray we use the following:

    myArray[2]

    The 2 in this example can be replaced by a variable to get the element of the array at the index of the variable's value.
