CPB-02 Functions

Press enter/return to start

Instructions

  1. Using the definition of moveAvatarLeft (line 19) as a guide, create the following functions:
    • moveAvatarRight
    • moveAvatarUp
    • moveAvatarDown
  2. Replace the moveAvatar commands with the appropriate newly created function.

Notes

  1. The asterisk (*) in line 20 represents the multiplication operator.
  2. The moveAvatarRight function will not be used.
  3. Be careful with your parentheses and curly braces. Every opening parenthesis and curly brace must have a matching closing one.
    • If you receive an error that says "Unexpected end of input", it's likely you have an opening curly brace without a matching closing curly brace.
    • If you receive an error that says "Unexpected token '}'" or "Unexpected token ')'", it's likely you have a closing curly brace or parenthesis without a matching opening one.
