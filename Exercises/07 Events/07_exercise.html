CPB-07 Events

Events allow our programs to respond when something happens. In this exercise we will enable our program to move the avatar when the user presses an arrow key.

Instructions

  1. After the moveAvatar function, add a function named keyDownHandler that accepts one parameter named eventInfo.
  2. Inside of the keyDownHandler function, create a variable named spacesToMove and initialize it to a value of 10.
  3. After the variable declaration, add 4 if/else if conditional blocks that execute based on the value of the eventInfo object's key property. The 4 values to test for are the following strings:
    • 'ArrowLeft'
    • 'ArrowRight'
    • 'ArrowUp'
    • 'ArrowDown'
    The code blocks will be populated in the next step.
  4. In each code block, call the appropriate moveAvatar function (moveAvatarLeft, moveAvatarRigft, etc.) passing spacesToMove as the parameter.
  5. In the initialize function (line 8), register the keyDownHandler function as an event listener for the 'keydown' event (see note below).

Notes

  • To register an event listener in Javascript, we use the addEventListener function of the globally-defined document object. For example, to register a function named mouseClickHandler to be called whenever the mouse is clicked, we can use the following:

    document.addEventListener('click', mouseClickHandler)

  • For more information on Events in Javascript, see the W3 Schools reference.
